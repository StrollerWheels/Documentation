Файл начала программы

Алгоритм:

  • Настройка ядра МК
  • Настройка переферии МК
  • Настройка межзадачного взаимодействия во freeRTOS
  • Проверка причины перезапуска МК
  • Запуск планировщика freeRTOS

Настройки ОСРВ:

  • Приоритет задач от младшего: TaskInterface -> TaskPhy -> TaskControl
  • Очереди между каждой парой задач
  • вытесняющая многозадачность
