Подробное описание

Start of the program

Algorithm:

  • Initialization of MCU core
  • Initialization of MCU Periphery
  • Initialization inter-task communication in freeRTOS
  • Checking the cause if MCU restart
  • Starting the scheduler of freeRTOS

Initialization freeRTOS:

  • Prioritize tasks from less to more: TaskInterface -> TaskPhy -> TaskControl
  • Queues between each pair of tasks
  • Preemptive multitasking
