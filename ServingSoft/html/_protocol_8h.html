Протокол информационнго взаимодействия

Протокол прикладного уровня взаимодействия между блоком управления и Пультом. Применяется как для обмена через USB, так и через Bluetooth.

Внимание
  • Протокол должен быть синхронизирован и на Пульте и в блоке управления
  • Необходимо учитывать, чтобы файл одинакого компилировался GCC C compiler, MinGW C++ compiler и MSVC
