Подробное описание

Задача интерфейсная

Задача ОСРВ freeRTOS с низшим приоритетом в системе. Задача обеспечивает взаимодйствие Блока управления с Потребителем либо через USB, либо через блютуз.

Файл содержит реализацию функций выполения команд от Потребителя.

Необходимо сделать:
Убрать лишний отладочный код, см. hfArray
