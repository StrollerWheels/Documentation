<strong><em>Описание проекта Пульт технологический</em></strong>

CppCheck Doxygen QT Creator
  • Проект создан для информационного взаимодействия пользователя с Блоком управления коляской. Протокол взаимодействия тут.
  • Проект разработан в среде QT Creator 5.0.2, версия qt 5.15.2. Пульт работает под ОС Windows 10 или позже.
  • Обмен данными через Bluetooth осуществляется через драйвер USB-Bluetooth в виде виртуального com-порта.
  • Необходимо разрешить устройствам Bluetooth обнаруживать этот компьютер в настройках Windows.
  • Для стабильной работы необходимо установить скорость виртуального com-порта (bluetooth) 115200 бод в настройках Windows.
  • В проекте испсользуется 4 формы для взаимодействия с пользователем:
    • настройки управления,
    • настройки соединения,
    • окно телеметрии,
    • дополнительные настройки.
  • Взаимодействие между объектами основано на qt механизме сигнал-слот.

<strong><em>Инструкция по применению</em></strong>

  1. Необходи1о выбрать настройки соединения через меню Настройки -> Настройки соединения
  2. Установить доплнительные настройки через меню Управление настройками -> Дополнительные настройки.
    Внимание
    Важно! Дополнительные настройки записываются и в RAM и во Flash.
  3. Выбрать настройки управления настройками и записать их в RAM, нажав кнопку Записать настройки в RAM электронного блока управления.
  4. Запись настроек во Flash, чтение настроек из Flash и RAM доступно через меню Управление настройками
  5. Внимание
    Внимание! При подаче питания Блок управление копирует настройки из Flash в RAM
  6. После выбора всех настроек нажать кнопку Перейти в окно телеметрии без записи настроек
  7. Управление в окне телеметрии с помощью кнопок на форме.

<em><strong>Выполняемые функции:</strong></em>

  • Установка коэффициентов регулятора
  • Выбор типа уставки: угол или угловая скорость, синусоида или константа
  • Выбор использовать регулятор или нет
  • Установка дополнительных параметров Период скважности Время дребезга контактов Частота выдачи телеметрии Частота вызова регулятора
  • Отображение телеметрии на графике
  • Запись телеметрии в файл
  • Проверка соединения с Блоком управления коляской (пингование)
  • Переподключение при отсутствии блютуз соединения
  • Создано системой 1.9.7
